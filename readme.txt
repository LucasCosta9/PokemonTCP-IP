Este programa foi feito em Java, é multithread e possui conexão TCP/IP, e seu objetivo é
simular uma batalha de pokemon semelhante ao jogo "Pokemon Fire Red", os autores do projeto
são:
Iderval da Costa e Silva Neto
Lucas Costa Bezerra (https://github.com/LucasCosta9)
Pedro Lucas Rangel Felix (https://github.com/plrf100)

Carregando as imagens
-Abra "Images.java" e troque todos os "X:\\UFPB\\Imagens" pelo endereço da pasta "Imagens"
que acompanha o código (Um Ctrl-F capaz de trocar todas as ocorrências ja resolve) 

Como executar o programa 
-Execute o "Server.java" e depois abra dois "Cliente.java", as ações precisam ser enviadas
pelo terminal de cada um dos clientes

1 = Atacar -> 1-4 = Escolher um dos ataques
5 = Selecionar troca de pokemon -> 1-5 = Escolher o pokemon que quer trocar
